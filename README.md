# Incomplete Prompt Jailbreaks (IPJ) — GitHub Pages

This repository hosts a **single-page, node-based** website for:

**Incomplete Prompt Jailbreaks in Large Language Models**  
Yeonjea Kim · Bumjin Park · Jaesik Choi

## Edit links (paper / dataset / code)

Open `assets/app.js` and set:

- `LINKS.paper.href`
- `LINKS.dataset.href`
- `LINKS.code.href`

## Local preview

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
